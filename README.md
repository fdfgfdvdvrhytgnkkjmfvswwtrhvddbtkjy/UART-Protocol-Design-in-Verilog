# UART-Protocol-Design-in-Verilog
**UART Protocol Design in Verilog:** A UART communication system implemented in Verilog HDL featuring a Baud Rate Generator, UART Transmitter (TX), and Receiver (RX) using FSMs. Supports 8N1 format (8-bit data, no parity, 1 stop bit) and verified through ModelSim loopback simulation.
